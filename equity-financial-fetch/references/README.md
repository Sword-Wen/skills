## 数据源详细说明

### 一、美股数据源

#### 1. StockAnalysis（SEC API）- 首选

**简介**：
- 本地工具，基于美国证券交易委员会（SEC）官方 REST API
- 提供最权威、最完整的美股财报数据

**数据覆盖**：
- 所有在美上市公司的财报数据
- 资产负债表、利润表、现金流量表
- 历史数据可追溯至公司上市时

**访问方式**：
```bash
# 命令行方式
python -m stock_analyzer AAPL --years 10

# Python API 方式
from stock_analyzer import StockAnalyzer
analyzer = StockAnalyzer()
result = analyzer.get_financial_indicators(ticker='AAPL', years=10)
```

**输出指标**：
| 指标 | 说明 | 单位 |
|------|------|------|
| 营业收入 | Revenue | 亿美元 |
| 归母净利 | Net Income | 亿美元 |
| 毛利率 | Gross Margin | % |
| 运营利润率 | Operating Margin | % |
| 净利率 | Net Margin | % |
| ROE | Return on Equity | % |
| ROA | Return on Assets | % |
| 净现比 | Operating CF / Net Income | 倍 |
| 经营净现金流 | Operating Cash Flow | 亿美元 |
| 资本开支 | Capital Expenditures | 亿美元 |
| 自由现金流 | Free Cash Flow | 亿美元 |
| 股东回报 | Dividends + Buybacks | 亿美元 |
| 有息负债 | Interest-bearing Debt | 亿美元 |

**优点**：
- 数据最权威（来自 SEC 官方）
- 更新及时
- 无访问限制

**缺点**：
- 仅支持美股
- 部分公司历史数据可能不完整

## 财务指标计算公式

### 盈利能力指标

| 指标 | 公式 | 说明 |
|------|------|------|
| 毛利率 | (营业收入 - 营业成本) / 营业收入 × 100% | 反映产品盈利能力 |
| 营业利润率 | 营业利润 / 营业收入 × 100% | 反映主营业务盈利能力 |
| 净利率 | 净利润 / 营业收入 × 100% | 反映最终盈利能力 |
| ROE | 净利润 / 股东权益 × 100% | 反映股东投资回报 |
| ROA | 净利润 / 总资产 × 100% | 反映资产利用效率 |

### 现金流指标

| 指标 | 公式 | 说明 |
|------|------|------|
| 净现比 | 经营现金流 / 净利润 | 反映利润质量 |
| 自由现金流 | 经营现金流 - 资本开支 | 反映可自由支配的现金 |

### 增长指标

| 指标 | 公式 | 说明 |
|------|------|------|
| 营收同比 | (本期营收 - 上期营收) / 上期营收 × 100% | 反映营收增长 |
| 净利润同比 | (本期净利润 - 上期净利润) / 上期净利润 × 100% | 反映利润增长 |

## 市场识别规则详解

### 美股识别规则

```
规则 1：纯字母代码
- 示例：AAPL、MSFT、GOOGL、TSLA
- 判断：代码仅包含字母（A-Z，不区分大小写）

规则 2：带后缀的代码
- 示例：BRK.B、BRK.A
- 判断：包含点号（.）分隔的代码
```

### A股识别规则

```
规则 1：6位数字代码
- 以 0 开头：深圳主板、中小板
- 以 3 开头：创业板
- 以 6 开头：上海主板

示例：
- 000001：平安银行（深圳主板）
- 300750：宁德时代（创业板）
- 600519：贵州茅台（上海主板）
```

### 港股识别规则

```
规则 1：4-5位数字代码
- 以 0 开头：普通港股
- 以 1 开头：普通港股
- 以 2 开头：普通港股

示例：
- 00700 或 0700：腾讯控股
- 09988 或 9988：阿里巴巴-SW
- 01810 或 1810：小米集团

注意：港股代码通常省略前导零
```

## 常见问题与解决方案

### Q1：美股数据获取失败

**可能原因**：
- 股票代码错误
- 网络连接问题
- SEC API 暂时不可用

**解决方案**：
1. 检查股票代码是否正确
2. 检查网络连接
3. 稍后重试
4. 切换到备选数据源（Yahoo Finance）

### Q2：A股/港股数据获取失败

**可能原因**：
- 股票代码格式不正确
- 数据源网站暂时不可访问
- 数据尚未更新

**解决方案**：
1. 确认股票代码格式正确
2. 尝试切换到备选数据源
3. 检查是否为财报披露期（数据可能延迟）

### Q3：部分指标数据缺失

**可能原因**：
- 公司未披露该指标
- 数据源未收录该指标
- 指标名称在不同市场有差异

**解决方案**：
1. 标注缺失的指标
2. 尝试从备选数据源获取
3. 使用相关指标进行估算

### Q4：数据单位不一致

**可能原因**：
- 不同市场使用不同单位
- 不同数据源的默认单位不同

**解决方案**：
- 美股：统一转换为亿美元
- A股：统一转换为亿元
- 港股：统一转换为亿港元
