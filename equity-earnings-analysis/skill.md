# Equity Earnings Analysis (财报跟踪分析)

## 核心定位与原则

### 分析核心
秉持巴菲特与芒格的投资体系，透过财务数字看背后的"生意本质"，评估企业的确定性与竞争地位。

### 三大核心原则

1. **真实性优先**：强调对非经常性损益（如税收调整、资产计提）的剥离，还原企业真实的经营现金流和净利润。

2. **前瞻导向**：不仅分析滞后指标（营收/利润），更需挖掘领先指标（如RPO、预收款、活跃用户）。

3. **数据驱动决策**：所有分析结论必须有数据支撑，关键数据需注明权威来源。

---

## 适用场景

- 对某家上市公司指定季度财报进行分析（如：2025年第三季度）
- 对某家上市公司指定年度财报进行分析（如：2025年年度）
- 跟踪已关注企业的最新经营表现
- 评估企业季度/年度业绩变化与驱动因素

## 使用说明

1. 提供待分析上市公司的名称或股票代码
2. 明确指定分析的财报类型和期间（如：2025Q3季报、2025年年报）
3. 如有特定关注点，可一并说明

## 报告结构概览

完整报告结构见 `references/templates/earnings-report-template.md`。

**报告章节**：
1. 整体经营表现（核心业绩陈述、经营趋势图表）
2. 细分业务拆解
   - 2.1 业务板块营收分析（总体概述+饼图）
   - 2.2 各业务逐一分析
   - 2.3 市场板块营收分析（如无明显市场分部可标注"不适用"）
3. 财务质量分析（费用、盈利、收益质量、财务风险、资本开支、股东回报）
4. 总结与展望
5. 估值

## 分析要点说明

### 1. 整体经营表现

**核心业绩陈述**：
- 营收：绝对金额、同比变化（百分比和增量）
- 归母净利润：优先使用 non-GAAP 数据，并说明与 GAAP 差异
- 一次性费用还原：涉及重大一次性费用时，需增加"真实净利润"补充陈述
- 年度视角：若本季度为财年最后一个季度，增加全年度数据

**经营趋势图表**（可选）：展示过去 10 个季度的营收和净利润趋势。

### 2. 细分业务拆解

**业务板块分析**：
- 给出各业务板块的营收、占比及经营结果概述
- 逐一分析各业务：营收数据、增长驱动力、护城河评估、未来增速预测

**市场板块分析**：
- 若无明显市场分部，可标注"不适用"
- 若有，则陈述各市场的营收数据及增长原因

### 3. 财务质量分析

> **重要说明**：本章节所有利润表及现金流指标均采用**截止本季度末的本财年口径（Year-to-Date, YTD）**。

**段落结构要求**：每个指标一个自然段，自然段开头以一句简短总结作为首句。

#### 3.1 费用支出分析
- 本财年累计（YTD）总费用支出
- 分项费用：研发费用、营销和销售费用、一般及行政费用
- 分析要点：费用率变化趋势、费用结构优化或恶化

#### 3.2 盈利能力分析（YTD口径）
- 毛利率、运营利润率、净利率
- ROE（年化）、ROA（年化）
- 分析要点：盈利能力变化趋势、毛利率变动驱动因素

#### 3.3 收益质量分析（YTD口径）
- 净现比 = 经营活动现金流净额 / 归母净利润
- 分析净现比高低的原因

#### 3.4 财务风险分析（截止季度末时点）
- 有息负债规模、净现金规模
- 利息覆盖倍数

#### 3.5 资本开支评估（YTD口径）
- 资本开支规模、原因和主要投向
- 自由现金流压力测试

#### 3.6 股东回报分析（YTD口径）
- 股息支付金额、股票回购金额
- 回购与SBC对比、股份稀释抵消效果

### 4. 总结与展望

本部分分为四个自然段落：

1. **概括总结**：总结财报表现，定性判断"护城河"（拓宽/稳定/受损）

2. **未来展望**：陈述未来增速与增长驱动力，数据来源于权威机构分析师一致预测（三个月内更新）

3. **市场担忧**：从逆向角度分析潜在问题与风险，以编号列表形式呈现

4. **下季度关注点**：列出下季度重点关注的指标和事件，以编号列表形式呈现

### 5. 估值

**预测模型选择**：

| 模型 | 适用企业类型 | 理想买点 |
|-----|------------|---------|
| 模型1 | 成长性企业（基于CAGR预测） | 合理估值的 50% |
| 模型2 | 周期性企业（席勒估值法） | 合理估值的 70% |

**估值计算**：

1. **归母净利润预测**：基于权威机构分析师一致性预测，直接叙述结论（不列出各机构分项数据）

2. **合理估值与理想买点**
   - 未来第三年合理估值 = 归母净利润 × 25
   - 理想买点 = 合理估值 × 50%（成长性）或 × 70%（周期性）

3. **机会成本测算**
   - 预期 CAGR 回报率 = (三年后合理估值 / 当前市值)^(1/3) - 1

## 语言风格与叙事特征

1. **客观冷静**：基于专业的、数据驱动的分析语境

2. **数据表达**：使用"同比大增XX%（或XX亿）至XX亿"的标准化描述

3. **叙述优先**：财务质量分析章节以自然段落叙述为主，每个子节开头以一句简短总结作为首句

4. **护城河视角**：在业务拆解时关注各业务的竞争壁垒

5. **图表可选**：Image占位符标注为"(可选)"，可根据需要决定是否插入图表

## 数据来源要求

### 美股财报数据获取
优先通过 `equity-financial-fetch` skill 获取 SEC API 官方数据。

### 财务数据
- **优先**：公司正式发布的财报数据（如美股采集自SEC.gov）
- 交易所公告

### 预测数据
- 分析师一致预测（最近三个月内更新）
- 明确标注预测来源和更新时间

---

## 分析师预测数据获取规范

### 内部分析流程（两步法）

**第一步：识别预测权威性TOP3机构**

评估维度：预测准确度、覆盖持续性、报告质量、市场影响力

数据来源：
- 国际：Bloomberg、Reuters、TipRanks、MarketBeat
- 国内：东方财富Choice、同花顺iFinD、朝阳永续

**第二步：一致性预测综合结论**

综合TOP3机构的预测数据，形成一致性预测结论，直接在报告中叙述。

### 报告呈现要求

- 直接叙述一致性预测结论
- 不分别陈述各机构具体数据
- 注明数据来源和更新时间
