# 参考资料目录

本目录用于存放投资分析报告生成所需的参考资料。

## 目录规划

```
references/
├── templates/           # 报告模板
│   └── report-template.md
├── data-sources.md      # 推荐数据源清单
├── industry-classification.md  # 行业分类参考
├── formulas.md          # 财务指标计算公式汇总
└── examples/            # 示例报告
```

## 文件说明

### templates/
存放报告输出的Markdown模板，提供标准格式参考。

### data-sources.md
记录推荐的数据获取渠道，包括：
- 财报数据源
- 分析师预测平台
- 行业研究机构
- 市场数据平台

### industry-classification.md
行业分类标准参考，便于进行同行对比分析。

### formulas.md
财务指标计算公式汇总，确保计算口径一致。

### examples/
优秀分析报告示例，供参考学习。
